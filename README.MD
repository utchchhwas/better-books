## Setup Instructions:
1. Make sure you have `python` and `npm` installed.
2. Clone the repo:
    ```
    git clone https://github.com/utchchhwas/GoodReads.git
    cd GoodReads
    ```
3. Install dependencies:
    ```
    pip install -r requirements.txt
    cd frontend && npm install
    ```
4. Run `react` server:
    ```
    npm start
    ```
5. Run `django` server:
    ```
    cd ..
    python manage.py runserver 
    ```
